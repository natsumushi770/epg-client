# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** デフォルトのTauriアイコンを、グロー発光グリーンのブロードキャストアンテナ＋「EPG」テキストのカスタムアイコンに置き換える。

**Architecture:** SVGソースファイルを作成し、Tauri CLIの `tauri icon` コマンドで全プラットフォーム・全サイズのアイコンを一括生成して `src-tauri/icons/` に上書きする。

**Tech Stack:** SVG, `@tauri-apps/cli` (v2) の `tauri icon` サブコマンド

---
